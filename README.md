# 🎬 Movie Listing App

A responsive movie listing web application built using React. Users can browse movies, search by title, and rate them using a star-based system.

---

## 🚀 Live Demo
👉 https://your-vercel-link.vercel.app

---

## ✨ Features

- 🎥 Display movie cards with image, title, genre, and year  
- ⭐ Star rating system (interactive)  
- 🔍 Search movies by name  
- 📱 Fully responsive design  
- 🎨 Clean and modern UI  

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript (ES6)
- CSS3
- Git & GitHub
